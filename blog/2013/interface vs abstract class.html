Interface vs Abstract Class

08 March 2013

Interfaces supports multiple inheritence where as abstract class does not.

Below is a summary of differences
S.noAbstract ClassInterface
1 Abstract Class can have methods with definition Interface cannot have any method with definition
2 Abstract Class can have variable fields Interface can only have constant fields
3 Abstract Class cannot be used for multiple inheritance Interface can be used for multiple inheritance
4 Abstract Class makes use of VTable for method mappings Interface do not have such capabilities
PS: This is before Java 8
